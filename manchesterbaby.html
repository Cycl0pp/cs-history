The Manchester Baby

The Manchester Baby

Dates: First run on June 21, 1948

Why Significant

The Manchester Baby, known formally as the Small-Scale Experimental Machine, was the first computer to store and run a program from memory. This was an integral step in developing modern computers, which perform the same task regularly.

History

This proof-of-concept machine was built from military surplus supplies and was meant to prove that programs could be stored and recalled electronically. It could store up to 1,024 bits, using one Williams-Kilburn tube. None of the original parts remain.
